The Table Tags

This is a practice file - it will not be turned in nor graded. It is for you to learn from and experiment with.

HTML allows you to create tables with the following tags:
table - starts a table
tr - starts a table row
td - starts a table cell within a row (think of td as table data)
th - used to create special table cells in the first row of a table, called table headers
caption - used to give a caption to a table
thead - used around one or more tr tags, to indicate that these rows are part of the table head and contain no data, just titles or explanatory information
tbody - used around one or more tr tags, to indicate that these rows contain the data portion of the table
tfoot - used around one or more tr tags, to indicate that these rows are part of the table foot.
