Inline Elements

Inline elements, unlike block elements, do not take up an entire row by themselves. Inline elements flow easily with text, and do not move automatically to the next line like the block elements. One thing that you should remember when using inline elements is that they should not contain block elements. Below we discuss some of the inline elements.

a Element

The a (anchor) element is to create anchor tags. Anchor tags can be used to create hyperlinks to other pages or to other locations in the same page. To create hyperlinks to external webpages (i.e., webpages that are not on your website) use the href attribute with the http: or https: protocol. If you are linking to webpage on the same website, the href value should be a relative path to that webpage. You can also link to files, like regular text files, pdf files, PowerPoint files, MS Word files, images, etc. When you do this, the web browser may try to display the contents of the file.

Another thing anchor tags can do is create a named location within a web page. You can then "jump" to that location with another anchor tag by using the # symbol.

The anchor tag can also be used to indicate a telephone number; for that you need to use the href attribute with the tel: protocol. Finally, it can be used to indicate an e-mail address; for that you need to use the mailto: protocol.

sup Element

The sup (superscript) element is used to enclose superscripts. Compare it to the sub element.

Examples using the sup element

A quadratic polynomial is a polynomial of degree 2 and has the following general formula:
p(x) = ax2 + bx + c
where a, b and c are real numbers, and a is not 0.

A cubic polynomial is a polynomial of degree 3 and has the following general formula:
p(x) = ax3 + bx2 + cx + d
where a, b, c, and d are real numbers, and a is not 0.

wbr Element

The wbr (word break) element is used to give the browser a hint that if it needs to break a word to continue text on the next line, it should break the word at the point indicated by the wbr element. It does not create a line break, but it creates a line break opportunity. Use it in words that are long, like some chemical names. For examples of places that have long names, see this Wikipedia page.

Deprecated elements

Deprecated elements are elements that are marked for elimination. When they are deprecated, it means at some future point they will no longer be supported. Below is a list of deprecated elements.

  • big - Was previously used to create text that is larger in size than regular text. It is somewhat surprising that <big> was declared deprecated, but <small> was not. Small was used to create text that is smaller than regular text.
  • s - this was previously known as the strike element. Like the del element, it drew a line through the middle of the text (not above, and not below the text, but through the middle). It was reminiscent of the editing that my English teacher would do on the homework I would submit.
  • u - Was previously used to create underlined text. Now that is done using CSS.
  • tt - This was the teletype element. It was used to display text in a monospaced font.
