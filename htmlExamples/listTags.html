The List Tags

This is a practice file - it will not be turned in nor graded. It is for you to learn from and experiment with.

HTML allows you to create three types of lists:
- the unordered list with the ul and li tags
- the ordered list with the ol and li tags
- the description list with the dl, dd and dt tags (HTML 4 used to call these definition lists)

The li (list item) tag is used in both the unordered and ordered lists. A description list consists a sequence of pairs of dt and dd tags. dt introduces a term, the dd provides a description for the term.

Below you will see some examples of each of these different types of lists. For information on how to style lists, the webpage https://www.w3schools.com/cssref/pr_list-style-type.asp is recommended. Please note that you can have a list within a list, and a list within a list within a list (that is, a list nested two levels deep), and so on. There is no limit.

The Unordered List

The ul element, together with the li (list item) element, is used to create unordered lists.

Examples using the ul element

Subjects I took as a freshman in college:
  • History I
  • History II
  • English
  • Business Math I
  • Business Math II

Here is the same list, but styled so as not to display the marker symbol of the line items. This is done by using the attribute style="list-style-type: none" on the ul element.

  • History I
  • History II
  • English
  • Business Math I
  • Business Math II

Here is the same list, but styled to use squares. This is done by using the attribute style="list-style-type: square" on the ul element.

  • History I
  • History II
  • English
  • Business Math I
  • Business Math II

Here is the same list, but styled to use circles. This is done by using the attribute style="list-style-type: circle" on the ul element.

  • History I
  • History II
  • English
  • Business Math I
  • Business Math II

The Ordered List

The ol element, together with the li (list item) element, is used to create ordered lists.

Examples using the ol element

My weekend chores list:
  1. Take wife to grocery store
  2. Wash the dog
  3. Mow the lawn
  4. Pay the car insurance bill online
In this next example, I use the CSS property list-style-type and use the lower-roman value; that is, I used style="list-style-type: lower-roman;" attribute on the ol element.
Steps to getting married:
  1. Buy an engagement ring (optional)
  2. Ask the woman to marry you. The rest of the list assumes she said yes.
  3. Buy a wedding ring
  4. Go to the courthouse with fiance, and get a marriage license
  5. Use home or rent an event center for the wedding
  6. Hire a priest, rabbi, or preacher to perform at wedding
  7. Find best man, and best woman.
  8. Have wedding ceremony, give bride the wedding ring.

The Description List

The dl (description list), dt, and dd elements are used to create description lists.

Examples using the dl element

Coffee
A bitter beverage with substantial amount of sweetner added, which I drink every morning, wakes me up.
Beer
A beverage my wife lets me drink only on weekends.
Tequila Sunrise
A intoxicating version of liquid courage I could drink when I was single. Drank often before asking a woman to dance.
