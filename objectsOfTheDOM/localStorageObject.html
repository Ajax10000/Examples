The LocalStorage Object

The window.localStorage object has the ability to store large amounts of data on the user's computer for long term use. The amount of data it can store depends on the device; it varies from 2MB to more than 100MB. The data is stored in key/value pairs, where the key and value are both strings. Using JavaScript, you can read, write, and delete data from the localStorage object.

As mentioned above, localStorage can only store string values. However, it will automatically convert numbers into Strings for you. Also, you can store objects by first "stringifying" them with JSON.stringify(), but later when you read it to get the original object you will have to use JSON.parse(). Properties

Methods

clear()

Arguments: 0

Call syntax:
window.localStorage.clear();

The clear method removes all items in the local storage. To remove just one item from localStorage, use the removeItem() method.

getItem()

Arguments: 1, the key whose corresponding value you want returned

Call syntax:
window.localStorge.getItem(key);

Given a key value, getItem() returns the corresponding value from the localStorage.

key()

Arguments: 1, the index of the key whose corresponding value you want returned.

removeItem()

Arguments: 1, the name of the key that you wish to remove

Call syntax:
window.localStorage(nameOfKey);

Given the name of a key in the localStorage, removeItem() removes they key and corresponding value from the localStorage object. To remove all items in localStorage, use the clear() method.

setItem()

Arguments: 2, the key (as a String) and corresponding value (as a String) you want stored

Call syntax:
window.localStorage.setItem(key, value); localStorage.setItem(key, value);

Given a key and its correponding value, setItem() stores the key, value pair in the localStorage object.
