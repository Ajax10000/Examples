The EventTarget Object

The Node object inherits from the EventTarget object.

Properties

Methods

dispatchEvent()

Arguments: 1, the Event object that is to be dispatched

Call syntax:
eventTarget.dispatchEvent(eventObjectToDispatch)

The dispatchEvent method returns a boolean value. It returns false if, at any point in the propagation of the event, the Event.preventDefault method was called. Otherwise, it returns true.

Given an Event object (created with the Document.createEvent() method), the dispatchEvent method will generate the Event specified as the argument.

addEventListener()

Arguments: 3, the last one is optional
The first argument is the name of the event we are adding a listener for. The second argument is a function that will be called when the event occurs. This function is called the event handler. The third argument is optional, and it is a boolean value. If true, the event is event handler is called in the capturing phase. If false, the event handler is called in the bubbling phase.

Call syntax:
eventTarget.addEventListener(eventName, eventHandlerFunction [, booleanValue]);
