Properties

Methods

createElement()

Arguments: 2
The first argument is a string that specifies the type of element to create (head, style, script, p, div, br, etc.). The second argument is optional. If given, the second argument is an ElementCreationOptions object.

Call syntax:
document.createElement(elementType, optionalElementCreationOptions)

Note that the createElement() method creates and returns an Element object, but does not add it to the webpage. To add the newly created element to the webpage, use the Node.appendChild() or the Node.insertBefore() methods.

createTextNode()

Arguments: 1
document.createTextNode(text)

The createTextNode method is used to create a text node. Text nodes represent the text that goes between the start and end tags of such elements like body, p, div, pre, nav, main, header, footer, aside, section, and article tags, among others.

getElementById()

Arguments: 1, a string value which is the id of an element

Call syntax:
document.getElementById(idValue)

Given an id, getElementById() will return an Element object whose id matches that of the given id. If element is found with the given id, getElementById() returns null.

getElementsByClassName()

Arguments: 1, a class name

Call syntax:
document.getElementsByClassName(className)

Given a class name, the method getElementsByClassName returns an HTMLCollection object containing all the elements it found with that class name.

getElementsByName()

Arguments: 1, a name

Call syntax:
document.getElementsByName(name)

Given a name, the getElementsByName returns a NodeList object containing all the elements it found whose name attribute matches the given name.

getElementsByTagName()

Arguments: 1, a tag name (like "p", "table", "tr", "hr", "div", etc.)

Call syntax:
document.getElementsByTagName(tagName)

Given a tag name, method getElementsByTagName returns a NodeList object containing all the nodes it found whose element types match that of the given tag name.

querySelector()

Arguments: 1, a string containing one or more selectors to match. The selectors can be elements to match, ids to match, or classes to match.

Call syntax:
document.querySelector(query)

The querySelector method returns an Element object which matches the given selector. Compare to the querySelectorAll, which returns a NodeList object.

Webpages on the Document.querySelector() Method

querySelectorAll()

Arguments: 1, a string containing one or more selectors to match. The selectors can be elements to match, ids to match, or classes to match.

Call syntax:
document.querySelectorAll(query)

The querySelectorAll method returns a NodeList object. The NodeList contains all elements that match the selectors in the argument given. Compare to the querySelector method, which returns an Element object. Examples are given below. 

        var matchesFound = document.querySelectorAll("div");

The example statement above will return a NodeList containing all div tags in the webpage. 

        var allTablesWithChemDataClass = document.querySelectorAll("table.chemData");

The example statement above will return a NodeList containing all table tags with the "chemData" class.
