The HTMLCollection Object

The HTMLCollection stores a collection of items in document order (that is, they appear in the HTMLCollection in the same order as they appear in the document).

Webpages on the HTMLCollection Object

  • https://developer.mozilla.org/en-US/docs/Web/API/HTMLCollection

Properties

length

Access: read only

The length property is an integer that indicates how many items there are in the HTMLCollection.

Methods

item()

Arguments: 1, an integer indicating the index of the item in the HTMLCollection to return.

Call syntax:
HTMLCollection.item(idx);

Given an index i, the item(i) method returns the item in the HTMLCollection at that index. If i is outside the range of valid index values for the collection, then item(i) returns null.

namedItem()

Arguments: 1, a string representing the id or name of the item in the HTMLCollection to return.

Call syntax:
HTMLCollection.namedItem(idOrName);

Given a string, HTMLCollection.namedItem() returns the element whose id matches the given string. If no element has the given id, it returns the element whose name matches the given string. If no element has a matching name, then it returns null.
