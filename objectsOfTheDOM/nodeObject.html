The Node Object

The Node object represents a generic node in the DOM tree. More specific objects represent specific types of elements in the DOM tree. Many DOM objects inherit from the Node object, including the Document and Element objects.

Properties

attributes

A list of attributes of the current node, if the current node has any.

nodeType

The nodeType property is a number, an integer actually, which indicates what type of object the node really is. See the table below.

Integer Value Corresponding Object Type
1 Element
2 Attr
3 Text
4 CDATASection
5 EntityReference (now obsolete)
6 Entity (now obsolete)
7 ProcessingInstruction
8 Comment
9 Document
10 DocumentType
11 DocumentFragment

textContent

The text content of a tag is the text that appears between the starting tag and the closing tag. If the current node has text content, this property holds it. Not all elements can have text context. Some elements that can not have text content are "hr", "br", "img", "input", and "meta".

Methods

appendChild()

Arguments: 1, a Node object to append.

Call syntax:
node.appendChild(childNode);

The return value of the appendChild method is the childNode that was appended.

The appendChild method is used to append a node as the last child element of a given node. An example is shown below. 

        var webpageBody = document.body;

        // Create a new Element
        var myNewFooterNode = document.createElement("footer");

        // Give it a background color
        myNewFooterNode.style.backgroundColor = #eee;

        // Add it as the last child of the body element
        webpageBody.appendChild(myNewFooterNode);

Like appendChild, the insertBefore method can also be used to insert an element into a webpage.

hasChildNodes()

Arguments: 0

Call syntax:
node.hasChildNodes();

The hasChildNodes method returns true if the given element has any child nodes, and returns false otherwise. An example is shown below. 

        <!DOCTYPE html>
        <html>
        <head>
        <meta charset="UTF-8">
        <title>Outerspace Aliens</title>
        </head>
        <body>
        <h1>All I know about outerspace aliens</h1>
        <-- I don't know anything about outerspace aliens -->
        <div id="content">
        </div>
        </body>

Given the webpage above, the following hasChildNodes method would return false. 

        var divElement = document.getElementById("content");
        if (divELement.hasChildNodes()) {
            console.log("It appears I know something about outerspace aliens");
        } else {
            console.log("Apparently I know nothing about outerspace aliens");
        }

insertBefore()

Arguments: 2 The first argument is the new child node to insert. The second argument is the node you want to insert your new node before.

Call syntax:
node.insertBefore(newChild, beforeChild);

Like insertBefore, the appendChild method can also be used to insert an element into a webpage.

removeChild()

Arguments: 1, the child node to remove from the parent node

Call syntax:
node.removeChild(childNodeToRemove);

Given a child node of a parent node, this method will remove the child node from the parent node.

replaceChild()

Arguments: 2
The first argument is the child node to replace. The second argument is the new node that will be the replacement.

Call syntax:
node.replaceChild(newChild, childToBeReplaced);
