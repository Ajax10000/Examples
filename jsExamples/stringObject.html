The String Object

The String object represents a string. This object contains many methods. Many of them are discussed below.

Several of the String object's methods are now deprecated. This means they will probably be eliminated in a future version of JavaScript. It is recommended that you not use these deprecated methods. The deprecated methods are:

  • anchor
  • big
  • blink
  • bold
  • fixed
  • fontcolor
  • fontsize
  • italics
  • link
  • small
  • strike
  • sub
  • sup

The deprecated methods are not discussed below.

Methods

charAt

The charAt method takes one parameter, an integer representing an index into the string. The string's indices go from 0 to (the length of the string - 1). For example the string "abc" is of length 3, and its indices go from to to 2.

The charAt method returns the character at a specific index of the string. Examples:
"abc".charAt(0) returns "a"
"abc".charAt(1) returns "b"
"abc".charAt(2) returns "c"

indexOf

Given a substring, the indexOf method returns an integer giving the index in the string at which the substring can be found. If the substring cannot be found in the string, indexOf returns -1. An example is shown below. 

    var animalsOfAfrica = "lions, cheetahs, leopards, elephants, giraffes, zebras, hyenas";
    var animalsOfAmerica = "jaguars, pumas, bobcat, bison, wolverines, beavers, pronghorn deer";
    var animal = prompt("Please enter the name of an animal");

    if (animal != null) {
        animal = animal.toLowerCase();
        if (animalsOfAfrica.indexOf(animal) != -1) {
            alert("That animal lives in Africa.");
        } else if (animalsOfAmerica.indexOf(animal) != - 1) {
            alert("That animal lives in North or South America");
        } else {
            alert("I'm not sure where that animal lives.");
        }
    }

repeat

Arguments: 1
The repeat method takes one parameter, a non-negative integer, between 0 and +Infinity. Providing a negative integer will generate a RangeError. If the integer provided generates a string that exceeds the maximum string length, a RangeError is generated. According to this reference, the maximum string length per ECMAScript 2016 is 253 - 1.

Call syntax:
string.repeat(nonNegativeInteger);

Given a non-negative integer n, the repeat method will repeat a string n times. Examples:
"a".repeat(3) returns "aaa"
"n".repeat(5) returns "nnnnn"
"ab".repeat(2) returns "abab"
"water".repeat(5) returns "waterwaterwaterwaterwater"

split

Arguments: 1
The split method takes a longer string, usually containing a character that is repeated in the string like a comma (,), semicolon (;), colon (:), dash (-), etc. The split method, given a string, will then return an array where each element is a substring of the original string, which has been split at each occurrence of the given argument. Examples are shown below. 

    var states = "Alabama, Alaska, Arizona, Arkansas";
    var arrayOfAStates = states.split(",");
After the above statements executes, arrayOfAStates will be an array of length 4 with arrayOfAStates[0] = "Alabama", arrayOfAStates[1] = "Alaska", arrayOfASttes[2] = Arizona, and arrayOfAStates[3] = "Arkansas".

Arguments: 0

The toLowerCase method takes no parameters. The toLowerCase method converts every character in the given string to lower case characters. Examples:
"Alphabet".toLowerCase() returns "alphabet"
"ALPHABET".toLowerCase() returns "alphabet"
"alPhabet".toLowerCase() returns "alphabet"
"alphabet".toLowerCase() returns "alphabet"

trim

Arguments: 0

The trim method takes no parameters. The trim method deletes every leading and trailing whitespace character in the string. That is, it deletes every whitespace character in the front of the string and the back of the string. What is a whitespace character? The whitespace charcters include spaces, tabs, carriage returns, new lines characters, vertical tab characters and the form feed characters. Examples:
" Alphabet ".trim() returns "Alphabet"
"ALPHABET".trim() returns "ALPHABET"
" Printer".trim() returns "Printer"
"television ".trim() returns "television"
