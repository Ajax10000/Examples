JavaScript vs jQuery

First off, let me start by saying that jQuery is a JavaScript library. This means it is JavaScript code, packaged in a manner that anyone can include it in their html page to use. jQuery is not a programming language like JavaScript is.

The purpose of jQuery is to make JavaScript easier to use. With that in mind, the following table will show you how you can do something in JavaScript, and then show you how to do the same thing using jQuery.

Task JavaScript jQuery
Get a reference to an existing HTML element by using an id var myElement = document.getElementById("myIdValue");
or
var myElement = document.querySelector("#myIdValue");

For more on document.getElementById(), see https://developer.mozilla.org/en-US/docs/Web/API/Document/getElementById.

For more on element.querySelector(), see https://developer.mozilla.org/en-US/docs/Web/API/Element/getElementsByClassName.

var myElement = $("#myIdValue");

See https://www.w3schools.com/jquery/jquery_selectors.asp. See also 05-Third-Party-APIs, activity 04-jQueryDrinkList.

Get a reference to an existing HTML element by using a class var myElements = document.getElementsByClassName("myClassName");
or
var myElement = document.querySelector(".myClassName");

Note that document.getElementsByClassName() returns an object of type HTMLCollection. You can access the elements of HTMLCollection like you would an array. See https://developer.mozilla.org/en-US/docs/Web/API/Document/getElementsByClassName

Unlike document.getElementsByClassName(), element.querySelector() returns one Element, the first element it finds with the given class. See https://developer.mozilla.org/en-US/docs/Web/API/Document/querySelector. See also 04-Web-APIs, activities 04-Ins_Setting-Attributes and 05-Stu_Setting-Attributes.

var myElement = $(".myClassName");

See https://www.w3schools.com/jquery/jquery_selectors.asp.

Create an element var myNewLiElem = document.createElement("li");

For more on element.createElement(), see https://developer.mozilla.org/en-US/docs/Web/API/Document/createElement. See also 04-Web-APIs, activities 06-Ins_Create-Append and 07-Stu_LookMaNoHTML

var myNewLiElem = $("<li>");

See https://api.jquery.com/jquery/#creating-new-elements. See also 05-Third-Party-APIs, 11-FridgeGame

Get the attribute of an element var myImgElement = document.getElementById("hotStuff");
var imgSrcAttr = myImgElement.getAttribute("src);

See https://developer.mozilla.org/en-US/docs/Web/API/Element/getAttribute.

var myImgElement = $("#hotStuff");
var imgSrcAttr = myImgElement.attr("src");

See https://api.jquery.com/attr/. See also 05-Third-Party-APIs, activities 03-jQueryGenerators and 11-FridgeGame.

Set the attribute of an element var myImgElement = document.getElementById("hotStuff");
myImgElement.setAttribute("src", "images/HotStuff.jpg");

See https://developer.mozilla.org/en-US/docs/Web/API/Element/setAttribute. See also 04-Web-APIs, activities 04-Ins_Setting-Attributes, 05-Stu-Setting-Attributes, and 07-Stu_LookMaNoHTML

var myImgElement = $("#hotStuff");
myImgElement.attr("src", "images/HotStuff.jpg");

See https://api.jquery.com/attr/. See also 05-Third-Party-APIs, activities 03-jQueryGenerators and 11-FridgeGame.

Removing the attribute of an element var myImgElement = document.getElementById("hotStuff");
myImgElement.removeAttribute("src");

See https://www.w3schools.com/jsref/met_element_removeattribute.asp.

var myImgElement = $("#hotStuff");
myImgElement.removeAttr("src");

See https://api.jquery.com/removeAttr/#removeAttr1.

Adding a CSS class to an element var myDivElem = document.createElement("div");
myDivElem.classList.add("myNewClass")

See https://developer.mozilla.org/en-US/docs/Web/API/Element/classList.

var myDivElem = $("<div>");
myDivElem.addClass("myNewClass");

See https://www.w3schools.com/jquery/html_addclass.asp

Set the text of an element var myParagraphElement = document.createElement("p");
myParagraphElement.textContent = "This is the content of my paragraph."

See https://www.w3schools.com/jsref/prop_node_textcontent.asp.

var myParagraphElement = $("<p>");
myParagraphElement.text("This is the content of my paragraph.");

See https://api.jquery.com/text/. See also 05-Third-Party-APIs, activity 03-jQueryGenerators and 11-FridgeGame.

Append a child element to a parent element (as the last child of the parent) var myDivElement = document.getElementById("myDivId");
myDivElement.appendChild(myParagraphElement);

See https://developer.mozilla.org/en-US/docs/Web/API/Node/appendChild. See 04-Web-APIs, activities 25-Stu_Render-Todos, 27-Stu_Complete-Todos, and 28-Stu-Local-Storage-Todos. Also see 05-Third-Party-APIs, activity 01-DOM_Manipulation.

var myDivElement = $("#myDivId");
myDivElement.append(myParagraphElement);

See https://api.jquery.com/append/. See also 05-Third-Party-APIs, activities activity 02-JSDrinkList, 03-jQueryGenerators.

Get the value of a form field (input, select, or textarea input field). var userNameField = document.getElementById("userName");
var userName = userNameField.value;

See https://www.w3schools.com/jsref/prop_text_value.asp.

var userNameField = $("#userName");
var userName = userNameField.val();

See https://api.jquery.com/val/.

To delete all children of a given parent node. var mainDiv = document.getElementById("mainDiv");
while(mainDiv.firstChild) {
mainDiv.removeChild(mainDiv.firstChild);
}

See https://developer.mozilla.org/en-US/docs/Web/API/Node.

var mainDiv = $("#mainDiv");
mainDiv.empty();

See https://api.jquery.com/empty/.
